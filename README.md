# Saudi AI Career Intelligence

An AI-powered labor market intelligence platform for analyzing AI, data, and technology career opportunities in Saudi Arabia.

## Overview

This project aims to analyze job postings, skills, companies, sectors, and career pathways in the Saudi AI and data market.

The goal is to transform unstructured career data into actionable insights for students, fresh graduates, and early-career professionals.

## Planned Features

- Job posting collection and cleaning
- Skill extraction from job descriptions
- AI/data role classification
- Market trend analysis
- Power BI dashboard
- RAG-based career assistant
- Personalized skill gap analysis

## Technologies

- Python
- Pandas
- NLP
- Power BI
- Streamlit
- LangChain
- Vector Databases
- Retrieval-Augmented Generation

## Status

Early-stage portfolio project. Data collection and dashboard design are in progress.
